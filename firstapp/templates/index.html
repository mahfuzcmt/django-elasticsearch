Requirements

  • Download and Install Java: https://www.java.com/en/download/
  • Download Elastic Search: https://www.elastic.co/downloads/elasticsearch
  • After Download Unzip folder, go to 'bin' folder and double click on 'elasticsearch.bat'
  • Server will be started at here: http://localhost:9200/
  • Install the package: https://github.com/sabricot/django-elasticsearch-dsl
  • pip install django-elasticsearch-dsl
